# Career Chatbot
## Backend setup
1. Create python environment
2. Run the commands below:
   ```
   cd backend
   pip install requirements.txt
   ```
3. Run mistral on Ollama:
   ```
   ollama run mistral
   ```
4. Run fastapi server:
   ```
   fastapi run main.py
   ```
## Frontend setup
1. Run the commands below:
   ```
   cd frontend
   npm install
   ```
2. Start the vite app:
   ```
   npm run dev
   ```
